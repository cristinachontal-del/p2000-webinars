# Pentagon 2000 Webinar Pages

Each HTML file in this repo is a webinar landing page hosted via GitHub Pages.

## Live URLs
Each file is accessible at:
`https://cristinachontal-del.github.io/p2000-webinars/[filename].html`

| File | Live URL |

| upstream-june-2026.html | https://cristinachontal-del.github.io/p2000-webinars/upstream-june-2026.html |

## How to add a new webinar
1. Upload the new HTML file to this repo
2. The live URL will be: `https://cristinachontal-del.github.io/p2000-webinars/your-filename.html`
3. Paste that URL into the Wix CMS `webinarPageUrl` field for the new entry
